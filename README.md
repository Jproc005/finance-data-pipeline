# Finance Data Automation Pipeline (Flagship Project)

A **one-click financial reporting pipeline** that converts raw CSV exports into a standardized, Excel-ready report for **non-technical business users**.

This project demonstrates **Python + SQL (SQLite) + automated reporting** with clean architecture, auditability, and user-friendly execution.

---

## What Problem It Solves

Many teams receive financial data exports that are inconsistent from source to source (different column names, formats, and cleanliness).  
This tool allows a user to drop CSV files into a folder and generate a clean Excel report automatically.
